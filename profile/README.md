# Axiom Odyssey

**Модпаки и моды для Minecraft на современной Java.**

</div>

---

Axiom Odyssey — независимая команда, делающая свою вселенную модов для Minecraft.
Наш ориентир — большие технологические 1.7.10-паки, но со своим кодом, своими решениями
и запуском на **актуальной Java (21–25)**, а не на устаревшей Java 8.

## Проекты

| Репозиторий | Что это |
|---|---|
| **[Axiom-Odyssey](https://github.com/AxiomOdyssey/Axiom-Odyssey)** | Основной модпак Minecraft 1.7.10 (Forge + LWJGL3), запускается на Java 21–25 |
| **AxiomLib** *(скоро)* | Собственная библиотечная основа для модов Axiom Odyssey |

## Принципы

- **Своя разработка.** Не клон существующих паков — своя идентичность и свой код.
- **Современная Java.** 1.7.10 без боли с Java 8.
- **Чистые лицензии.** Каждый сторонний мод — с проверенной лицензией и указанием авторства.
